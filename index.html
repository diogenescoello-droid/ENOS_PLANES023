BASE 01 · Ecosistema digital ENOS F01-F07

Seleccione su rol, llene los formularios asignados, actualice la base, copie el prompt, use el GPT para generar el plan de acción y registre el respaldo final en la carpeta de cierre de ciclo.

1. Seleccione el tipo de actor

Cada actor ingresa a un espacio propio. La información se mantiene ordenada por código de caso, institución y formulario.

Área GAD

Llene los formularios asignados, manteniendo el mismo código de caso en cada registro.

Rol activo: GAD

Formularios asignados

Use el mismo código de caso en todos los formularios. No mezcle casos o instituciones.

Cómo llenar sin errores

  1. Defina un solo código de caso para el plan o expediente.
  2. Use ese mismo código en F01 a F07.
  3. Registre datos concretos y verificables.
  4. Cuando exista evidencia, agregue archivo, foto o enlace.
El código de caso permite que el GPT lea únicamente la información del caso activo.

Actualizar base y preparar prompt

Este módulo va antes de entrar al GPT. Actualiza Kobo, genera el paquete y copia el prompt técnico.

Listo. Actualice la base cuando existan nuevos registros de Kobo.

Qué hace este módulo

  • Actualizar base: sincroniza Kobo con Google Sheets.
  • Ver casos: muestra casos disponibles y estado.
  • Generar y copiar prompt: arma el paquete técnico para GPT.
No ingrese al GPT sin copiar primero el prompt. Ese bloque contiene la información estructurada del caso.

Usar GPT, descargar plan y cerrar ciclo

Pegue el prompt copiado, permita que GPT revise suficiencia, genere el plan de acción en Word y cargue el respaldo final.

1. Pegar prompt

Ingrese al GPT y pegue el prompt generado.

2. Descargar plan

Solicite el Word institucional, descárguelo y revise el contenido técnico.

3. Firmar y subir

Firme el plan y súbalo a la carpeta de cierre de ciclo.

Primero genere y copie el prompt en el módulo anterior. Luego pegue aquí el enlace del GPT institucional una sola vez y presione “Guardar y abrir GPT”. En adelante el navegador recordará el enlace.

Cierre del ciclo

Después de descargar y firmar el plan, use la carpeta de cierre configurada en el Dashboard.

Cierre de ciclo

  1. Descargue el plan generado en Word.
  2. Revise formato, datos, conclusiones, recomendaciones y anexos.
  3. Gestione firma o validación institucional.
  4. Suba el documento firmado a la carpeta de cierre.
  5. Actualice seguimiento mensual en F07.
