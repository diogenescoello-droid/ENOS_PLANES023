Seleccione el actor que usará el sistema

El sistema muestra formularios y acciones según el rol seleccionado. La Coordinación Zonal mantiene la trazabilidad del ciclo completo.

Flujo operativo

Seleccione un actor para iniciar el trabajo.

Actor no seleccionado

Formularios Kobo disponibles

Use los formularios según el tipo de actor y el momento del ciclo. Todos alimentan la base consolidada F01-F07.

Uso práctico

  1. Seleccione el actor.
  2. Abra el formulario correspondiente.
  3. Use el mismo código de caso en todos los formularios.
  4. Al finalizar, vuelva al módulo “Actualizar y prompt”.
El código de caso es la llave para integrar información de GAD, MTT, Grupo Técnico y CZ5.

Actualizar base y generar prompt GPT

Sincronice Kobo, seleccione el caso y genere el prompt técnico para el asistente ENOS Zona 5.

Sistema listo. Seleccione un caso o actualice la base.

Control de suficiencia

  • F01 identifica el caso y los puntos críticos.
  • F02 incorpora infraestructura esencial expuesta.
  • F03-F07 completan cartografía, acciones, preparación, capacidades y seguimiento.
  • Si faltan datos, el GPT debe declarar limitaciones y brechas.

GPT institucional y cierre práctico

Ingrese al asistente técnico ENOS Zona 5, pegue el prompt generado y descargue el plan de acción desde el GPT.

Cierre del ciclo en esta plataforma: el usuario genera el plan con el GPT y descarga el archivo final. La firma, envío por Quipux, archivo institucional o socialización se gestionan fuera de esta página, según el procedimiento interno de cada institución.

Flujo de cierre práctico

1. Generar

Pegue el prompt en el GPT institucional y solicite el plan de acción.

2. Revisar

Verifique datos, conclusiones, recomendaciones y responsabilidades antes de usarlo oficialmente.

3. Descargar

Descargue el archivo final desde el GPT en Word, PDF o el formato que corresponda.

Criterio operativo

  1. La página no registra cierres administrativos.
  2. No se exige carpeta de cierre de ciclo.
  3. No se utiliza F08.
  4. El cierre funcional es la descarga del plan desde el GPT.
  5. La trazabilidad formal queda en los procedimientos institucionales externos.

Bibliografía e insumos de apoyo

Acceso directo a normativa, lineamientos, cartografía, fuentes institucionales y respaldos técnicos que acompañan la construcción del plan.

Cartografía

F03 alimenta mapas, capas, fuentes, formatos y limitaciones.

Insumos técnicos

F01-F06 consolidan amenazas, infraestructura, acciones y capacidades.

Seguimiento

F07 documenta avance mensual, evidencias y escalamiento.

Repositorio técnico

Este botón abre directamente la carpeta oficial de Drive con normativa, lineamientos e insumos de apoyo para la elaboración del plan.

El cierre práctico del ciclo se realiza mediante descarga directa del plan desde el GPT. La biblioteca queda como repositorio de consulta técnica.
