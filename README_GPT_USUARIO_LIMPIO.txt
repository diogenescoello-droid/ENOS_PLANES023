ENOS Guayas 2.1 - Ajuste del módulo GPT

Este paquete elimina del módulo público GPT el bloque de descarga de archivos de configuración interna.

La plataforma mantiene los archivos administrativos en docs/configuracion/ para que el equipo responsable pueda actualizar el GPT desde Editar GPT > Configure > Instructions.

Para el usuario final, el módulo GPT queda orientado a:
- Abrir el GPT Técnico ENOS.
- Confirmar solicitante, institución, rol, producto, territorio y filtros de registros.
- Usar prompts operativos mediante botones de copiar.
- Evitar mezclar registros de distintos GAD, parroquias o Mesas Técnicas.

Carga recomendada en GitHub:
1. Descomprimir este ZIP.
2. Entrar a la carpeta descomprimida.
3. Seleccionar todo el contenido interno.
4. Arrastrar a la raíz del repositorio ENOS_PLANES022.
5. Commit changes.
