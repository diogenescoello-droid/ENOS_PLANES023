PAQUETE LIMPIO ENOS GUAYAS 2.1 PARA GITHUB PAGES

Repositorio destino:
https://github.com/diogenescoello-droid/Plan_Accion_ENOS02

Modo de carga recomendado:
1. Entrar al repositorio Plan_Accion_ENOS02.
2. Seleccionar todos los archivos y carpetas de este paquete, no la carpeta contenedora.
3. Arrastrarlos al área de carga de GitHub.
4. Confirmar que GitHub muestre archivos en estas rutas:
   - index.html en la raíz.
   - assets/site.css y assets/site.js.
   - data/config.json.
   - docs/Manual_uso_Plataforma_ENOS_Guayas_2_0.docx.
   - docs/Plantillas_maestras_ENOS_Guayas_2_0.xlsx.
   - docs/matrices/matriz_maestra_enos_2_1.xlsx.
   - docs/formularios/...
   - docs/insumos_sprea/...
   - docs/cartografia/...
5. Hacer clic en Commit changes.
6. Esperar 2 a 5 minutos y abrir:
   https://diogenescoello-droid.github.io/Plan_Accion_ENOS02/

Contenido del paquete:
- HTML completo de la plataforma ENOS Guayas 2.1.
- Enfoque corregido: GAD provinciales/cantonales remiten planes; GAD parroquiales y Mesas Técnicas aportan insumos.
- Burbujas flotantes de ayuda por módulo.
- Visor ArcGIS: https://arcg.is/iGSLu0
- Proyecto Google Earth: https://earth.google.com/earth/d/1FN-JQLdUzfxbABJKH3GPmdS57EZeHe4h?usp=sharing
- Biblioteca normativa, SPREA, manual, plantillas, matriz, formularios y cartografía base.
- Botones Drive redactados como solicitud de acceso, no como carpeta pública.

Nota importante:
No se incluyen presentaciones PPTX pesadas dentro de GitHub, porque GitHub Web limita cargas grandes. Las presentaciones SPREA se abren o solicitan desde Drive.
