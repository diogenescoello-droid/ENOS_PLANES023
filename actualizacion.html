Sincronización inmediata · Kobo → Sheets → GPT

Actualizar base consolidada y preparar paquete GPT

Use este módulo cuando un técnico haya llenado formularios y no se desea esperar la próxima actualización automática. El botón ejecuta la actualización inmediata y entrega un texto listo para copiar al GPT ENOS.

Uso interno: la actualización ejecuta un Web App institucional de Google Apps Script. Si solicita inicio de sesión, use la cuenta autorizada del dominio institucional.

Actualización inmediata

Ejecuta el ciclo completo: Kobo → hojas RAW → CONSOLIDADO_PLANES → DASHBOARD → SEGUIMIENTO_GAD → paquete GPT.

Resultado esperado: se abrirá una pantalla con el paquete generado y el botón “Copiar paquete para GPT”.

Actualizar por cantón

Use esta opción cuando quiera preparar un paquete filtrado, por ejemplo Playas, Daule, Durán o Lomas de Sargentillo.

El filtro no borra datos; solo limita el paquete que se copia al GPT.

Leyenda operativa

La base consolidada viva se actualiza mediante sincronización programada Kobo → Google Sheets → CONSOLIDADO_PLANES → DASHBOARD. Durante la fase de socialización y aplicación, los cortes oficiales deberán actualizarse manualmente con el botón de esta página antes de descargar base, revisar indicadores o preparar un plan en GPT.

Formulario Kobo lleno → botón Actualizar base y preparar paquete GPT → copiar paquete → pegar en GPT ENOS → revisar borrador técnico

Ruta recomendada para el usuario autorizado

  1. Verifique que el técnico haya enviado el formulario desde Kobo.
  2. Presione Actualizar base y preparar paquete GPT.
  3. Espere la pantalla de Apps Script y revise el número de registros incluidos.
  4. Presione Copiar paquete para GPT.
  5. Abra el GPT ENOS, pegue el paquete y solicite generación o actualización del Plan de Acción Territorial en formato SNGR.
