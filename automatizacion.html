De Kobo al consolidado, dashboard y borrador técnico

El flujo operativo actual conecta Kobo con Google Sheets mediante Apps Script: formularios → hojas RAW → CONSOLIDADO_PLANES → DASHBOARD. La generación de planes con GPT se mantiene como borrador técnico sujeto a revisión.

Estado actual: sincronización Kobo activa, consolidado automático y dashboard generado desde la base viva. El documento final aún requiere revisión técnica, validación institucional y remisión por autoridad competente.

Ciclo de actualización vigente

  1. Registrar o actualizar datos en Kobo.
  2. Abrir la base consolidada viva.
  3. Ejecutar ENOS Sync → 7. Ejecutar ciclo completo ENOS.
  4. Revisar CONSOLIDADO_PLANES.
  5. Revisar DASHBOARD.
  6. Usar la base para reportes, reuniones o borradores técnicos.

Control de seguridad y alcance

Acceso

La base viva debe permanecer restringida a usuarios autorizados por contener datos técnicos y operativos.

Revisión técnica

Los registros consolidados quedan como pendiente de revisión hasta que el equipo técnico confirme consistencia, pertinencia y trazabilidad.

Producto final

GPT genera insumos o borradores, no actos administrativos ni planes oficiales sin aprobación.
